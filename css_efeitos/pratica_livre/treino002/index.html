Aprendendo a usar Displays no CSS

Exemplos de Display (Block, Inline, Inline-Block e None)

Display Block

O display block exibe um elemento como um elemento de bloco. Começa em uma nova linha e ocupa toda a largura.

Display inline

Display inline exibe um elemento como um elemento embutido. Quaisquer propriedades de altura e largura não terão efeito.

Div do display inline
Div do display inline
Div do display inline

Display inline-block

Display inline-block exibe um elemento como um contêiner de bloco de nível embutido. O próprio elemento é formatado como um elemento embutido, mas você pode aplicar valores de altura e largura

Display none

O elemento display none é completamente removido
